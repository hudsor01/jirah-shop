**requireAdmin tests:**

2. Throws "Unauthorized: not authenticated" when getUser returns error
3. Throws "Unauthorized: not authenticated" when getUser returns null user

**sanitizeRedirect tests (pure function, no mocks):**
6. Returns the path when starting with single slash ("/dashboard")
7. Returns "/" for protocol-relative URL ("//evil.com")
8. Returns "/" for absolute URL ("https://evil.com")
9. Returns "/" for URL with backslash trick ("/\\evil.com")
10. Returns "/" for empty string (doesn't start with /)
11. Returns "/" for path starting with no slash ("dashboard")
12. Preserves query parameters ("/search?q=test")
13. Preserves hash fragments ("/page#section")

**sanitizeSearchInput tests (pure function, no mocks):**
14. Removes commas from input ("a,b" → "ab")
15. Removes parentheses ("foo(bar)" → "foobar")
16. Removes backslashes ("foo\\bar" → "foobar")
17. Removes closing braces ("foo}" → "foo")
18. Escapes percent signs ("50%" → "50\\%")
19. Escapes underscores ("foo_bar" → "foo\\_bar")
20. Preserves normal text ("hello world" → "hello world")
21. Handles combined injection attempt ("a,b(c)\\d}%e_f" → "abd\\%e\\_f")

Write tests for sanitizeRichHTML and sanitizeText. These use real isomorphic-dompurify (no mock needed — jsdom provides DOM).

**sanitizeRichHTML tests:**
1. Preserves allowed tags: h1, h2, h3, p, strong, em, a, ul, ol, li, img, table, blockquote, pre, code
2. Preserves allowed attributes: href, target, rel on anchors; src, alt, width, height on images
3. Strips script tags: `<script>alert('xss')</script>` → removed entirely
4. Strips event handlers: `<div onclick="alert('xss')">` → `<div>`
5. Strips style attributes: `<p style="color:red">text</p>` → `<p>text</p>`
6. Strips iframe tags: `<iframe src="evil.com"></iframe>` → removed
7. Strips class/id attributes: `<div class="foo" id="bar">` → `<div>`
8. Handles nested dangerous content: `<p><script>xss</script>safe</p>` → `<p>safe</p>`
9. Preserves complex valid HTML: table with thead/tbody/tr/td, figure/figcaption
10. Strips form tags: `<form action="evil"><input></form>` → content preserved, tags stripped
11. Handles empty input: "" → ""

**sanitizeText tests:**
12. Strips all HTML tags: `<p>hello</p>` → "hello"
13. Strips nested HTML: `<div><p><b>text</b></p></div>` → "text"
14. Preserves plain text: "hello world" → "hello world"
15. Strips script tags: `<script>alert('xss')</script>` → ""
16. Handles entities: `<p>&amp;</p>` → "&"
17. Handles empty input: "" → ""
