<task type="auto">
  <name>Task 1: Install DOMPurify and create sanitization library</name>
  <files>package.json, lib/sanitize.ts</files>
  <action>
1. Install dependencies:
   ```bash
   bun add isomorphic-dompurify
   bun add -d @types/dompurify
   ```

2. Create `lib/sanitize.ts` with two exported functions:

   `sanitizeRichHTML(html: string): string` — For blog content and product descriptions (rich text from admin editor):
   - Use `DOMPurify.sanitize()` with explicit `ALLOWED_TAGS` and `ALLOWED_ATTR` configuration
   - ALLOWED_TAGS: `h1, h2, h3, h4, h5, h6, p, br, strong, em, b, i, u, s, a, ul, ol, li, blockquote, pre, code, img, table, thead, tbody, tr, th, td, hr, span, div, figure, figcaption, sup, sub`
   - ALLOWED_ATTR: `href, target, rel` (on anchor tags), `src, alt, width, height` (on img tags)
   - No `style`, `class`, `id` attributes allowed (per CONTEXT.md locked decision)
   - No `iframe, script, object, embed, form, input` tags (enforced by omission from allowlist)

   `sanitizeText(text: string): string` — For plain text fields (reviews, contact form, names):
   - Strip ALL HTML tags using `DOMPurify.sanitize()` with `ALLOWED_TAGS: []` and `ALLOWED_ATTR: []`
   - Returns plain text only

   Import pattern: `import DOMPurify from "isomorphic-dompurify";`

3. Verify the module exports correctly:
   ```bash
   npx tsc --noEmit lib/sanitize.ts
   ```
